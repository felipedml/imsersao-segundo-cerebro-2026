# Suas entregas ficam aqui

Esta é a pasta onde o seu clone e os robôs **salvam o que produzem para você**: rascunhos de artigo, projeto de pesquisa, revisão de literatura, pareceres, slides, apps…

Você não precisa mexer aqui — os robôs organizam sozinhos. Volte quando quiser reabrir ou continuar um trabalho.

> Dica: cada trabalho pode virar um arquivo próprio. Se quiser, peça ao seu clone para "salvar isto em meu-projeto com um nome claro".
